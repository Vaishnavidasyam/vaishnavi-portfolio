# 🌌 Vaishnavi Portfolio

An ultra-modern, futuristic personal portfolio website built using React.js, Vite, Tailwind CSS, Framer Motion, and GSAP.

Designed to showcase my skills, projects, internships, and frontend engineering experience with premium UI/UX, glassmorphism aesthetics, smooth animations, and AI-inspired visuals.

---

## 🚀 Live Demo

🔗 https://vaishnavi-portfolio-orp1.vercel.app

---

## ✨ Features

- ⚡ Modern React + Vite architecture
- 🎨 Futuristic dark UI with glassmorphism
- 🌌 Animated gradients & glowing blobs
- 🌀 Smooth scrolling using Lenis
- 🎥 Framer Motion & GSAP animations
- 📱 Fully responsive design
- 💡 Interactive hover effects
- 📊 Animated skills progress bars
- 🧠 AI-inspired portfolio aesthetics
- 🌐 Modern section transitions
- 📂 Project showcase cards
- 📧 Contact section with social links

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/vaishnavi-portfolio.git
```

Move into project folder:

```bash
cd vaishnavi-portfolio
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
